Este repositório é referente ao curso da formação BI e DataWarehouse utilizando Power BI e SQL Server realizado pela Alura, O Curso Ensina desde a construção dos bancos de dados utilizando as tabelas fatos e dimensões através do Visual Studio, até a construção dos Cubos Olaps a serem utilizados para o Analysis Services, além de mostrar como realizar consultas MDX dentro do SQL Server Studio e finalmente construir e publicar os dashboards pelo Power BI.
